# LUMIÈRE — Women's Fashion Website

A beautiful, multi-theme fashion e-commerce website built with pure HTML, CSS & JavaScript.

## 🚀 Live Demo

After enabling GitHub Pages, your site will be live at:
`https://YOUR-USERNAME.github.io/YOUR-REPO-NAME`

## ✨ Features

- 5 stunning colour themes (Terracotta, Midnight, Emerald, Blush, Obsidian)
- Shopping cart with add/remove/quantity controls
- Wishlist functionality
- Product quick-view modal
- Animated hero section
- Fully responsive design

## 📁 Files

- `index.html` — The complete website (single file, no dependencies needed)

## 🌐 How to Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** → **Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**
5. Your site will be live in 1–2 minutes!
